# Dream Sequences

As a player, I want to experience surreal dream sequences when resting, so that the game can deliver lore, foreshadowing, and unique gameplay through an alternate reality lens.

## Details

- Dreams trigger randomly when performing a long rest (20% chance)
- Dream types:
  - **Prophetic**: shows a distorted vision of a future quest event (foreshadowing)
  - **Memory**: replays a key moment from the player's backstory in a playable flashback
  - **Nightmare**: combat encounter in a warped dreamscape against shadow versions of enemies
  - **Puzzle**: abstract logic puzzle in a surreal environment (reward: Wisdom XP or insight)
  - **NPC communication**: a dead NPC or distant ally reaches out through dreams (LLM dialogue)
- Dream maps use inverted colors and distorted layouts
- Items found in dreams occasionally persist on waking (rare dream artifacts)
- Recurring dreams hint at the main storyline's direction
- A dream journal records all dreams for later review

## Acceptance Criteria

- [ ] Dreams trigger during long rests with appropriate probability
- [ ] All dream types render with distinct visuals
- [ ] Dream artifacts occasionally persist
- [ ] Dream journal records all experienced dreams
