# Map Annotations

As a player, I want to place custom annotations and markers on my map, so that I can remember important locations, plan routes, and track points of interest.

## Details

- Place colored markers on any explored tile
- Add text notes to markers (e.g., "Locked door - need iron key")
- Marker categories: quest, danger, loot, NPC, custom
- Markers visible on both the minimap and full map view
- Remove or edit markers at any time
- Auto-markers for quest objectives and discovered locations

## Acceptance Criteria

- [ ] Player can place and remove map markers
- [ ] Text notes attach to markers
- [ ] Markers display on minimap and full map
- [ ] Auto-markers appear for quests and discoveries
