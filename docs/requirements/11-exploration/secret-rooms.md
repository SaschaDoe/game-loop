# Secret Rooms

As a player, I want to discover hidden secret rooms in dungeons, so that thorough exploration is rewarded with exclusive loot and lore.

## Details

- Secret rooms are hidden behind walls that look normal but can be walked through or opened
- Detection methods: high Perception stat auto-detects nearby, Search action examines adjacent walls, certain spells reveal secrets
- Secret rooms contain: treasure chests, lore inscriptions, rare items, NPC hideouts, shortcuts
- Each dungeon level has 0-2 secret rooms
- A subtle visual hint exists for attentive players (slightly different wall character)
- Some secret rooms are connected to quest content

## Acceptance Criteria

- [ ] Secret rooms exist behind hidden walls
- [ ] Multiple detection methods work correctly
- [ ] Secret rooms contain valuable rewards
- [ ] Visual hints are present but subtle
