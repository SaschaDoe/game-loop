# Fast Travel

As a player, I want to fast-travel between previously discovered locations, so that I don't have to tediously walk through cleared areas repeatedly.

## Details

- Fast travel unlocks between discovered towns and waypoints
- Requires a specific item (travel stone) or ability to use
- Fast travel consumes in-game time (proportional to distance)
- Random encounter chance during fast travel (ambush!)
- Cannot fast-travel while in combat, in a dungeon, or while encumbered
- Travel animation shows the path on the overworld map
- Some locations are only reachable on foot (no fast travel point nearby)

## Acceptance Criteria

- [ ] Discovered locations appear as fast-travel options
- [ ] Travel consumes appropriate in-game time
- [ ] Random encounters can interrupt fast travel
- [ ] Restrictions (combat, dungeon, encumbrance) are enforced
