# Language Barriers

As a player, I want some NPCs or creatures to speak in foreign or ancient languages, so that communication is a challenge that rewards investing in language skills or finding translators.

## Details

- Some races/factions speak their own language: Elvish, Dwarvish, Orcish, Ancient Tongue
- Untranslated dialogue appears as garbled/encoded text
- Player can learn languages by: studying books, taking lessons from NPCs, using translation items
- Partial language knowledge shows partially translated text
- Some quest-critical inscriptions require specific language knowledge
- Translator NPCs or magic items can help bridge the gap

## Acceptance Criteria

- [ ] Foreign language dialogue renders as untranslated text
- [ ] Learning a language progressively reveals translated text
- [ ] Translation items and NPC translators function correctly
- [ ] Language-locked content is accessible once the language is learned
