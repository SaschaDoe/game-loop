# Combat Log

As a player, I want a detailed combat log showing all actions and outcomes, so that I can understand exactly what happened each turn.

## Details

- Log displays: attacker, target, action, damage/healing, status effects applied
- Color-coded messages: red for damage taken, green for healing, yellow for player attacks
- Scrollable history of the last 50 messages
- Toggle between compact (one-line) and detailed (multi-line) views
- Log includes enemy AI decisions: "Goblin moves toward you", "Skeleton casts fireball"

## Acceptance Criteria

- [ ] All combat actions appear in the log
- [ ] Messages are color-coded by type
- [ ] Log is scrollable and retains history
- [ ] Enemy actions are narrated in the log
