# Magic Spells

As a player, I want to cast magic spells that deal damage, heal, buff, or debuff, so that I have versatile tactical options beyond physical attacks.

## Details

- Spells cost mana to cast; mana regenerates slowly each turn
- Spell schools: Fire, Ice, Lightning, Shadow, Holy, Nature
- Area-of-effect spells hit multiple tiles (fireball, chain lightning)
- Buff spells: increase stats temporarily (haste, shield, strength)
- Debuff spells: weaken enemies (slow, blind, poison)
- Healing spells restore HP to self or allies
- Spells are learned via skill trees, scrolls, or NPC teachers
- Casting requires Intelligence check; failure wastes mana with no effect

## Acceptance Criteria

- [ ] Spells consume mana correctly
- [ ] AoE spells affect the correct tile area
- [ ] Buffs and debuffs apply and expire correctly
- [ ] Spell learning from multiple sources works
