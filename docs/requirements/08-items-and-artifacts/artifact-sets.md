# Artifact Sets

As a player, I want themed artifact sets where collecting multiple pieces grants powerful set bonuses, so that I have long-term collection goals across the whole game.

## Details

- 6-8 artifact sets, each with 3-5 pieces scattered across different areas:
  - **Crown of the Storm King** (3 pieces: crown, scepter, mantle): 2-piece: +20% lightning damage, 3-piece: automatic chain lightning on every attack
  - **Shadow Assassin's Regalia** (4 pieces: mask, cloak, gloves, boots): 2-piece: +3 stealth, 3-piece: backstab from any direction, 4-piece: kills in stealth don't break stealth
  - **Dragon Slayer's Arsenal** (3 pieces: greatsword, shield, amulet): 2-piece: +30% damage to dragons, 3-piece: dragon breath immunity
  - **Healer's Devotion** (4 pieces: staff, robe, ring, circlet): 2-piece: +50% healing power, 3-piece: heals splash to nearby allies, 4-piece: auto-resurrect once per combat
  - **Chrono Weaver's Instruments** (3 pieces: hourglass, chronometer, monocle): 2-piece: +1 action per turn, 3-piece: Rewind ability free once per combat
  - **The Berserker's War Paint** (5 body modification tattoos): progressive rage damage bonus up to +100% at full set
- Set pieces have individual value too (not useless until complete)
- Set tracker in the inventory shows which pieces are found and which are missing with vague location hints
- Sets span multiple game areas — early pieces are findable, final pieces require endgame content

## Acceptance Criteria

- [ ] Set bonuses activate at correct piece counts
- [ ] Individual pieces have standalone value
- [ ] Set tracker shows progress and hints
- [ ] Set pieces are distributed across game progression
