# Armor System

As a player, I want to equip armor in multiple slots that reduces incoming damage, so that I can balance protection and mobility.

## Details

- Armor slots: head, chest, legs, boots, gloves, cloak
- Armor types: cloth, leather, chainmail, plate
- Heavier armor = more damage reduction but lower dodge chance and movement penalty
- Class restrictions: mages can only wear cloth/leather, warriors can wear all types
- Armor has durability that degrades in combat and can be repaired at blacksmiths
- Set bonuses: wearing a complete matching set grants extra benefits

## Acceptance Criteria

- [ ] All armor slots are equippable
- [ ] Damage reduction calculates correctly per armor type
- [ ] Movement and dodge penalties apply for heavy armor
- [ ] Set bonuses activate when full set is equipped
