# Seasonal World Changes

As a player, I want the game world to cycle through seasons that affect the environment, so that the world feels dynamic and different times offer different challenges.

## Details

- Four seasons: Spring, Summer, Autumn, Winter (cycle every 500 in-game turns)
- Spring: flowers bloom, rivers flood (new water tiles), animals more active, herbs plentiful
- Summer: longer days, drought (some water tiles dry up), fire hazard in forests
- Autumn: harvest festivals in towns, falling leaves visual effect, mushrooms spawn in forests
- Winter: snow covers terrain, rivers freeze (walkable ice tiles), blizzards reduce visibility, some monsters hibernate
- Seasonal NPCs: traveling merchants appear only in specific seasons
- Seasonal crafting ingredients: snowberries (winter only), sunpetals (summer only)
- Season indicator in the HUD

## Acceptance Criteria

- [ ] Seasons cycle at the correct interval
- [ ] Map tiles change visually per season
- [ ] Seasonal gameplay effects are active
- [ ] Season-exclusive content spawns correctly
