# Settings Menu

As a player, I want a settings menu to configure controls, display, and gameplay preferences, so that I can tailor the experience to my liking.

## Details

- Controls: rebindable keys for all actions
- Display: font size, color theme (classic, high contrast, colorblind modes), viewport size
- Gameplay: auto-pickup items toggle, combat speed, message display duration
- Audio: volume controls for music, sound effects, ambient (if audio is implemented)
- LLM settings: API key input, model selection, response length preference
- Reset to defaults option
- Settings persist across sessions

## Acceptance Criteria

- [ ] All settings are adjustable and take effect immediately
- [ ] Key rebinding works for all game actions
- [ ] Colorblind modes are functional
- [ ] Settings persist after closing and reopening the game
