# Main Menu

As a player, I want an ASCII art main menu when I start the game, so that the game has a polished first impression with clear navigation options.

## Details

- ASCII art title banner for the game name
- Menu options: New Game, Continue, Load Game, Settings, Credits, Quit
- Continue loads the most recent save
- New Game flows into character creation
- Animated ASCII background (flickering torches, swaying dungeon scene)
- Version number displayed in the corner

## Acceptance Criteria

- [ ] All menu options are functional
- [ ] ASCII art title renders correctly
- [ ] Continue loads the latest save
- [ ] Menu is navigable with keyboard
