# Tutorial and Onboarding

As a new player, I want a gentle tutorial that teaches me the game mechanics progressively, so that I'm not overwhelmed and can learn at my own pace.

## Details

- Tutorial is the first dungeon: a guided experience with contextual tips
- Step-by-step introduction of: movement, combat, items, inventory, NPC dialogue, quests
- Tips appear as messages when the player first encounters a mechanic ("Press 'i' to open inventory")
- Tutorial NPC guide: a friendly ghost who explains things with humor and patience (LLM-powered)
- Tip system can be toggled off in settings for experienced players
- Progressive disclosure: advanced mechanics (crafting, enchanting, factions) introduced later when relevant
- "Did you know?" tips on loading screens with random gameplay hints
- First town acts as a safe sandbox: no enemies, friendly NPCs, all basic shops available
- Skippable: experienced players can skip the tutorial entirely from the main menu

## Acceptance Criteria

- [ ] Tutorial introduces all core mechanics in order
- [ ] Contextual tips trigger on first encounter with each mechanic
- [ ] Tutorial is skippable
- [ ] Tips are toggleable in settings
