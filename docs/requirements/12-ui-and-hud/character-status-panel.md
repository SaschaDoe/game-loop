# Character Status Panel

As a player, I want a detailed character status panel showing all my stats, equipment, and effects, so that I always know my character's current state.

## Details

- Display: HP/MaxHP, Mana/MaxMana, all ability scores, armor rating, damage range
- Show equipped items in each slot with stats
- List active buffs and debuffs with remaining duration
- Current XP and progress to next level
- Gold count and carry weight
- Toggle-able with a hotkey (e.g., 'c' for character sheet)
- Panel overlays the map without covering it entirely (side panel or popup)

## Acceptance Criteria

- [ ] All stats display accurately
- [ ] Equipment slots show current gear
- [ ] Active effects display with durations
- [ ] Panel is toggle-able and doesn't block gameplay
