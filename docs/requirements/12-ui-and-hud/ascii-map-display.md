# ASCII Map Display

As a player, I want a clear, color-coded ASCII map display, so that I can read the game world at a glance and distinguish between terrain, entities, and items.

## Details

- Consistent color coding: walls (dark grey), floor (grey), player (yellow), enemies (red variants), NPCs (green/blue), items (magenta/cyan)
- Smooth scrolling when the player moves near the viewport edge
- Viewport centered on the player with configurable size
- Tile legend accessible at any time
- Zoom levels: normal, zoomed in (larger characters), zoomed out (minimap-like)
- Animations for combat hits, spell effects, explosions (brief ASCII art flashes)

## Acceptance Criteria

- [ ] Map renders with consistent color coding
- [ ] Viewport scrolls smoothly with player movement
- [ ] Zoom levels are switchable
- [ ] Combat animations play correctly
