# Accessibility Options

As a player with accessibility needs, I want options to adjust the game's visual and interaction design, so that the game is playable regardless of my abilities.

## Details

- Colorblind modes: deuteranopia, protanopia, tritanopia (symbol-based differentiation in addition to color)
- High contrast mode: bold outlines, brighter colors, larger characters
- Screen reader support: text descriptions of map state, entities, and actions
- Adjustable game speed: slow down turn processing for players who need more time
- Keyboard-only play (no mouse required) - already inherent in ASCII roguelike design
- Font size scaling from small to extra large
- Reduced motion: disable ASCII animations

## Acceptance Criteria

- [ ] Colorblind modes use distinct symbols in addition to color changes
- [ ] High contrast mode is visually effective
- [ ] Screen reader descriptions are accurate and helpful
- [ ] All options are independently toggleable
