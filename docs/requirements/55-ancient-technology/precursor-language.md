# Precursor Language

As a player, I want to decipher the Precursor writing system to unlock deeper lore, hidden commands, and ancient terminal access, so that linguistic skill is rewarded in ancient ruins.

## Details

- **Language Discovery**: Precursor text found on walls, terminals, data tablets, and weapon inscriptions; initially appears as unreadable glyphs; each glyph encounter teaches a fragment
- **Deciphering Progress**: language comprehension tracked as percentage (0-100%); each Precursor text found adds 2-5% based on length; INT modifier speeds learning; at key thresholds, new capabilities unlock:
  - 10%: understand basic warning signs (trap warnings, danger levels)
  - 25%: read simple terminal commands (open door, activate light, basic map)
  - 50%: understand data logs (lore entries, historical records, personal journals)
  - 75%: operate complex terminals (weapon crafting, security overrides, teleporter controls)
  - 100%: read the Precursor Master Archive (reveals the civilization's fate, unlocks the most powerful Precursor items and locations, ultimate lore dump)
- **Rosetta Stone Items**: rare translation devices found in Precursor libraries; each grants +15% comprehension instantly; 3 exist in the world
- **Terminal Interaction**: at sufficient comprehension, terminals become interactive; choose commands from a Precursor menu; wrong commands (partial comprehension guessing) may trigger security, lock the terminal, or corrupt data
- **Hidden Messages**: some Precursor texts are encrypted beyond standard language; require both high comprehension and a cipher key (found separately); these reveal the most secret locations and artifacts
- **Teaching Others**: at 75%+ comprehension, teach Precursor language to companion NPCs; companions can then help translate in parallel (faster ruin exploration)
- **Precursor Voice**: at 100%, hear ghostly Precursor voices in ruins; they narrate what happened in each room; provides atmosphere and quest-relevant information

## Acceptance Criteria

- [ ] Language comprehension tracks correctly as percentage
- [ ] Capabilities unlock at correct comprehension thresholds
- [ ] Terminal interaction requires sufficient comprehension level
- [ ] Rosetta Stone items grant correct instant comprehension bonus
- [ ] Hidden messages require both comprehension and cipher keys
