# Forbidden Archives

As a player, I want certain libraries to contain sealed forbidden sections with dangerous knowledge protected by magical wards and guardians, so that the most powerful scholarly content requires adventuring to access.

## Details

- **Forbidden Section Access**: found in major city libraries, ancient monasteries, wizard towers, and Precursor archives; entrance sealed by: magical wards (dispel or bypass), physical locks (lockpicking or key quest), knowledge tests (answer riddles), or reputation gates (head librarian permission at max scholar reputation)
- **Ward Types**:
  - Alarm ward: alerts guardians without blocking entry; must be disarmed (Arcana check) or endured (fight guardians)
  - Pain ward: deals damage when touched; bypassed with antimagic or specific counter-spell
  - Memory ward: erases knowledge gained inside if you leave without permission; must find the ward anchor and destroy it
  - Maze ward: the archive interior shifts and loops; navigation puzzle to reach the deep stacks
- **Forbidden Content**:
  - Necromancy texts: learn raise dead, soul trap, create undead; illegal in most kingdoms; guards confiscate if found in inventory
  - Forbidden histories: truth about kingdom founding (built on genocide), deity origins (gods were once mortal), faction secrets; reveals hidden quest lines and dialogue options
  - Dimensional formulae: portal creation instructions, summoning circles for extraplanar beings; using them attracts attention from powerful entities
  - True Names: documents containing true names of demons, fey lords, and ancient beings; knowing a true name gives power over that entity (auto-succeed on Persuasion/Command)
  - Weapon Blueprints: plans for weapons of mass destruction (plague formulas, world-ender artifacts, city-destroying spells); possessing these makes every faction want you dead or allied
- **Archive Guardians**: bound elementals, animated armor, spectral librarians, or golem constructs that protect the archives; some can be reasoned with (convince them you're authorized); others attack on sight
- **Knowledge Corruption**: reading too much forbidden content in one session (3+ tomes) causes "Knowledge Overload" (-2 WIS for 50 turns, nightmares, hearing whispers); manageable but unpleasant

## Acceptance Criteria

- [ ] All ward types function with correct bypass mechanics
- [ ] Forbidden content grants correct knowledge and quest access
- [ ] True Names provide correct power over named entities
- [ ] Archive guardians respond appropriately to authorization attempts
- [ ] Knowledge Overload triggers at correct reading threshold
