# Speed Run Mode

As a player, I want a speed run mode with a visible timer tracking real-time completion, so that I can challenge myself and compete for the fastest main story completion.

## Details

- Activatable from New Game settings
- Real-time timer displayed in the HUD (hours:minutes:seconds)
- Timer pauses: during menus, settings, and dialogue input (only gameplay time counts)
- Timer splits: automatic split tracking for major milestones (each dungeon cleared, boss killed, act completed)
- Leaderboard: local and exportable (share your time as a screenshot or code)
- Speed run optimizations: skip cutscenes, faster text scroll, auto-loot
- Ghost replay: after completing a run, watch a turn-by-turn replay at high speed
- Categories: Any% (just finish), 100% (all quests, all bosses, full bestiary), Pacifist (zero kills)
- Seed locking: specify a world seed so runs are comparable

## Acceptance Criteria

- [ ] Timer tracks real gameplay time accurately
- [ ] Split times record at correct milestones
- [ ] Speed run settings (skip, auto-loot) function
- [ ] Multiple categories are trackable
