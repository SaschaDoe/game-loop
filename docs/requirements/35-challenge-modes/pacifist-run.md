# Pacifist Run

As a player, I want every encounter in the game to be solvable without killing, so that a complete pacifist playthrough is achievable and rewarded.

## Details

- Every enemy encounter has a non-lethal resolution:
  - Negotiate surrender through LLM dialogue (Charisma check)
  - Use sleep/charm spells to bypass without killing
  - Flee successfully
  - Use environmental tools to trap or redirect enemies
  - Pay bribes or offer food to hungry creatures
  - Solve the underlying problem (bandits rob because their village is starving — feed the village)
- Bosses have a diplomatic/puzzle solution alongside the combat one
- Kill counter tracked: displayed at end of game
- Zero-kill completion unlocks: "Peacemaker" title, unique pacifist-only equipment, alternate ending
- NPCs notice your pacifism: "I've heard you've never taken a life. Remarkable."
- Challenge: harder than combat — requires higher stats, more gold, creative solutions

## Acceptance Criteria

- [ ] Every mandatory encounter has a non-lethal path
- [ ] Kill counter tracks accurately
- [ ] Zero-kill rewards and ending are accessible
- [ ] Pacifist solutions require meaningful gameplay engagement
