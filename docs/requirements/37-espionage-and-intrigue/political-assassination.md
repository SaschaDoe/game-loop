# Political Assassination

As a player, I want quests involving the assassination or protection of political figures, so that the game has high-stakes political thriller content.

## Details

- Assassination quests: offered by rival factions, shadowy NPCs, or the player's own spy network
- Target types: faction leaders, corrupt nobles, tyrant kings, traitorous allies
- Planning phase: scout the target's schedule, identify guards, find entry points, choose method
- Methods:
  - **Direct combat**: fight through guards, confront the target (loud, risky, dramatic)
  - **Poison**: slip poison into food/drink during a social event (alchemy skill, stealth)
  - **Sniper**: ranged attack from a hidden position during a public appearance
  - **Frame job**: make it look like another faction did it (espionage + deception)
  - **Diplomacy**: convince the target to abdicate or flee (Charisma, only for non-lethal resolution)
- Protection quests: reverse — defend a target from assassination attempts
  - Investigate threats, identify the assassin, set up defenses, confront the plot
- Consequences: assassination changes faction leadership, alters world politics, affects reputation massively
- Moral weight: assassinating an innocent (even if ordered) causes heavy evil alignment shift

## Acceptance Criteria

- [ ] Multiple assassination methods are viable
- [ ] Planning phase provides useful recon
- [ ] Protection quests are playable as the inverse
- [ ] World state changes reflect the assassination outcome
