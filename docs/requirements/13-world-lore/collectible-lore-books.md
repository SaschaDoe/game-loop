# Collectible Lore Books

As a player, I want to find lore books scattered throughout the world, so that I can read them page-by-page and uncover the world's hidden history and secrets.

## Book Placement

- Books found in containers, on bookshelves, in treasure chests, on corpses, in libraries
- Some books placed in dungeon levels, some in settlements
- Rare books hidden in secret areas

## Book Content

- Each book has a title, author, and multi-page text content
- Content ties into Aethermoor lore: creation myths, faction histories, god legends, cautionary tales, prophecies
- Some books are short (1-2 pages), some longer (5-8 pages)
- Hidden clues about the true nature of the Ascended gods woven in subtly — never explicit exposition

## Book Reader UI

- Full-screen overlay in ASCII aesthetic
- Shows current page text with page number ("Page 3 of 7")
- A/D or Left/Right arrow to flip pages
- ESC to close reader and return to inventory
- Book title displayed at top

## Book Collection

- Reading a book from inventory opens the page-by-page reader
- Books are inventory items (take up one of 12 slots)
- Books can be stored in containers for later reading

## Acceptance Criteria

- [ ] Books are findable in the game world
- [ ] Books can be picked up into inventory
- [ ] Book reader displays pages with navigation
- [ ] Book content contains actual Aethermoor lore
- [ ] Books can be read from inventory
- [ ] Books can be stored in and retrieved from containers
