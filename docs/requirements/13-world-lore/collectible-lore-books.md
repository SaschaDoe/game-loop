# Collectible Lore Books

As a player, I want to find lore books scattered throughout the world, so that I can read them to learn about the world's history, cultures, and secrets.

## Details

- Books found on shelves, in treasure chests, on corpses, in libraries
- Categories: history, bestiary, geography, biographies, fiction, spell theory
- Reading a book adds it to a permanent collection in the menu
- Some books contain coded hints for puzzles or secret locations
- Rare books are quest items sought by collector NPCs
- Completing a book collection for a category grants a lore-related bonus
- Books are readable with full text content (multiple pages)

## Acceptance Criteria

- [ ] Books are findable throughout the game world
- [ ] Book content is readable with full text
- [ ] Collection tracking works correctly
- [ ] Collection completion grants bonuses
