# PvP Arena

As a player, I want to fight other players in a dedicated PvP arena, so that I can test my build against real opponents in a fair, structured setting.

## Details

- Arena accessible from any major town
- Matchmaking by character level (within 3 levels)
- Arena maps are symmetrical and purpose-built for fair combat
- Win/loss record tracked with a leaderboard
- Rewards for winning: arena tokens exchangeable for exclusive gear
- Spectator mode: watch ongoing matches
- No item loss on death in the arena (friendly combat)
- Best-of-3 format with a pre-match preparation phase

## Acceptance Criteria

- [ ] Arena is accessible and matchmaking finds opponents
- [ ] PvP combat resolves correctly
- [ ] Win/loss tracking and leaderboard function
- [ ] Arena rewards are purchasable with tokens
