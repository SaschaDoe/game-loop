# Shared World Events

As a player, I want periodic world events that all players can participate in simultaneously, so that the multiplayer experience has moments of shared excitement.

## Details

- Timed events announced server-wide: "A dragon has appeared in the Northern Wastes!"
- Events include: world boss raids, treasure hunts, faction wars, seasonal festivals
- Players converge on the event location and cooperate or compete
- Event rewards scale with participation
- Events change the world state temporarily (new NPCs, decorations, enemy spawns)
- Event schedule visible in-game calendar
- Solo players can participate alongside others or attempt events alone (scaled difficulty)

## Acceptance Criteria

- [ ] Events trigger on schedule and are announced to all players
- [ ] Event content is unique and engaging
- [ ] Rewards scale with participation
- [ ] Events affect world state during their duration
