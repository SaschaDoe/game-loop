# Quest Journal

As a player, I want a quest journal that tracks all active, completed, and failed quests, so that I can keep track of my objectives and progress.

## Details

- Categorized tabs: Active, Completed, Failed
- Each quest entry shows: title, description, objectives (with checkboxes), rewards, quest giver
- Active quests have a tracker showing next objective
- Map markers for quest-relevant locations
- Journal updates automatically as quest state changes
- Player can add personal notes to quest entries
- Search/filter functionality for finding specific quests

## Acceptance Criteria

- [ ] All quests appear in the journal with correct status
- [ ] Objectives update in real-time as progress is made
- [ ] Map markers point to quest locations
- [ ] Notes can be added to entries
