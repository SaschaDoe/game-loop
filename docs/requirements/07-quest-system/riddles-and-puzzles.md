# Riddles and Puzzles

As a player, I want to encounter riddles, logic puzzles, and environmental puzzles in dungeons and quests, so that gameplay challenges my brain, not just my combat stats.

## Details

- Riddle doors: a guardian or inscription poses a riddle; correct answer opens the way
- Logic puzzles: lever sequences, pressure plates, pattern matching
- Environmental puzzles: push blocks to create paths, redirect water, light torches in order
- LLM-powered riddle master NPCs that can generate unique riddles on the fly
- Hints available for a cost (gold, item, or relationship expenditure)
- Intelligence stat provides subtle hints (highlighting relevant details)
- Failed puzzle attempts may trigger traps or alert enemies

## Acceptance Criteria

- [ ] Riddles accept free-text answers evaluated by LLM
- [ ] Logic puzzles have clear mechanics and solvable solutions
- [ ] Environmental puzzles interact with the map/physics system
- [ ] Hints are available through various means
