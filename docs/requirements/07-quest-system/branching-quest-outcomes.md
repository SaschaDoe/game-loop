# Branching Quest Outcomes

As a player, I want my choices during quests to lead to different outcomes, so that my decisions feel meaningful and the world reflects my agency.

## Details

- Key quests have at least 2-3 resolution paths (e.g., fight, negotiate, betray)
- Outcomes affect: NPC relationships, faction standing, available future quests, world state
- Some outcomes are morally ambiguous with no clear "right" answer
- NPCs remember which path you chose and react accordingly
- Certain quest chains branch entirely based on a pivotal early decision
- The world visually changes based on outcomes (a saved village prospers; a destroyed one is ruins)

## Acceptance Criteria

- [ ] Multiple resolution paths are available for branching quests
- [ ] Outcomes produce distinct world state changes
- [ ] NPCs reference the player's choices in future dialogue
- [ ] Visual world changes reflect quest outcomes
