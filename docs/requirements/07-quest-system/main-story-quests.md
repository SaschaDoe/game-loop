# Main Story Quests

As a player, I want a compelling main storyline with a chain of quests, so that my adventure has purpose and narrative drive beyond random dungeon crawling.

## Details

- Multi-act story arc: discovery, rising conflict, climax, resolution
- Main quests introduce key characters, locations, and the central conflict
- Player choices affect story direction (at least 3 branching paths leading to different endings)
- Main quest markers on the map guide the player but don't force linearity
- Story adapts to player's class, race, and backstory for personalized dialogue
- Estimated 15-20 main story quests forming the core narrative

## Acceptance Criteria

- [ ] Main quest chain is completable from start to finish
- [ ] Branching choices lead to meaningfully different outcomes
- [ ] Quest markers guide the player
- [ ] Story references player's character creation choices
