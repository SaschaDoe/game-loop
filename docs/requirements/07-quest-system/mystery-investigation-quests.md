# Mystery Investigation Quests

As a player, I want to solve murder mysteries and detective-style investigations, so that quests challenge my deductive reasoning and reward thorough exploration.

## Details

- Investigation quests: a crime has been committed, player must find the culprit
- Clue system: examine crime scenes for physical evidence (footprints, weapon, notes)
- Interview suspects via LLM dialogue — ask questions, detect inconsistencies, read body language
- Evidence journal: collected clues are logged and can be cross-referenced
- Multiple suspects with motives and alibis; some are red herrings
- Accusing the wrong person has consequences (innocent jailed, real criminal escapes)
- Wisdom and Intelligence stats help notice hidden clues and detect lies
- Examples: murder at the tavern, stolen artifact from the temple, poisoned well in the village, missing person last seen entering the dungeon

## Acceptance Criteria

- [ ] Investigation quests present clues and suspects
- [ ] LLM dialogue supports interrogation and deduction
- [ ] Evidence journal tracks collected clues
- [ ] Wrong accusations have consequences
