# Requirements

This folder contains all epics and user stories for the ASCII RPG project.

## Structure

Each subfolder represents an **epic** (a large feature area). Inside each epic folder are **user story** markdown files. Every user story file begins with a standard user story sentence:

> As a [role], I want [goal], so that [benefit].

## Epics Overview

| Epic | Description |
|------|-------------|
| 01-character-creation | Race, class, stats, backstory, and appearance |
| 02-world-generation | Procedural maps, biomes, towns, weather, seeds |
| 03-game-modes | Observer mode, normal play mode, spectating |
| 04-combat-system | Turn-based combat, spells, status effects |
| 05-npc-system | NPC routines, personalities, factions, merchants |
| 06-dialogue-system | LLM-powered conversations, persuasion, rumors |
| 07-quest-system | Main quests, side quests, riddles, branching outcomes |
| 08-items-and-artifacts | Weapons, armor, legendaries, cursed items, crafting |
| 09-monsters-and-enemies | Monster types, bosses, behaviors, loot |
| 10-character-progression | Leveling, skill trees, reputation, achievements |
| 11-exploration | Fog of war, secrets, traps, fast travel |
| 12-ui-and-hud | ASCII display, menus, save/load, accessibility |
| 13-world-lore | Creation myth, factions, lore books, prophecies |
| 14-survival-mechanics | Hunger, rest, environmental dangers |
| 15-multiplayer | Co-op, PvP arenas, shared worlds |
| 16-technical-requirements | Serverless, Netlify, SvelteKit static, CI/CD, PWA, testing |
| 17-non-functional-requirements | Performance, mobile responsive, offline, security, a11y, i18n |
