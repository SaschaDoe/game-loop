# Requirements

This folder contains all epics and user stories for the ASCII RPG project.

## Structure

Each subfolder represents an **epic** (a large feature area). Inside each epic folder are **user story** markdown files. Every user story file begins with a standard user story sentence:

> As a [role], I want [goal], so that [benefit].
