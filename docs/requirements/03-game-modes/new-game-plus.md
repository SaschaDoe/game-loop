# New Game Plus

As a player, I want a New Game+ mode after completing the main story, so that I can replay with my powered-up character and experience harder content and alternate story paths.

## Details

- Unlocks after completing the main story for the first time
- Carry over: character level, skills, equipment, gold, bestiary, codex
- Reset: quest progress, NPC relationships, map exploration, world state
- NG+ changes:
  - All enemies are 50% stronger (HP, damage, new abilities)
  - New enemy variants appear (elite versions of common monsters with NG+ prefix)
  - Alternate story branches available that weren't in the first playthrough
  - New legendary artifacts only findable in NG+
  - Hidden boss unlocked after the final boss (true ending)
  - NPCs have different dialogue acknowledging the player's power ("You seem... different")
- NG++ and beyond: each successive cycle adds another 25% difficulty, stacking
- Leaderboard: track how many NG+ cycles completed with character stats

## Acceptance Criteria

- [ ] NG+ is accessible after main story completion
- [ ] Carryover items and stats persist correctly
- [ ] Enemy scaling applies for each NG+ tier
- [ ] NG+-exclusive content is only available in NG+ runs
