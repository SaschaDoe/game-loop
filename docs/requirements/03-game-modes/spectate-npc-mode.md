# Spectate NPC Mode

As a player, I want to select any NPC and follow their perspective, so that I can understand their daily life, thoughts, and relationships from their point of view.

## Details

- From observer mode, click on any NPC to "follow" them
- Camera locks to the NPC and follows their movement
- A thought bubble / inner monologue feed shows what the NPC is thinking (LLM-generated)
- See the NPC's stats, inventory, relationships, and current goal
- Can switch between NPCs freely
- Can exit back to free-camera observer mode at any time

## Acceptance Criteria

- [ ] Player can select any NPC to follow
- [ ] Camera follows the selected NPC
- [ ] NPC thoughts are displayed via LLM generation
- [ ] NPC stats and relationships are visible
