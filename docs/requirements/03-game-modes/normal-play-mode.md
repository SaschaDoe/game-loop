# Normal Play Mode

As a player, I want a standard play mode where I control my character and interact with the world, so that I can experience the RPG through direct gameplay.

## Details

- Turn-based movement on the ASCII map
- Full interaction with NPCs, items, enemies, and environment
- Player actions drive the game loop (move, attack, use items, talk, rest)
- World responds reactively to player actions
- HUD displays health, mana, status effects, minimap, and messages
- Game can be saved and loaded at any point
- Death results in game over screen with option to reload or restart (unless permadeath)

## Acceptance Criteria

- [ ] Player can move, fight, talk, and use items
- [ ] HUD displays all relevant character information
- [ ] Save and load functions work correctly
- [ ] Game over screen appears on death with appropriate options
