# Creative Mode

As a player, I want a creative mode where I can build, explore, and experiment without combat pressure or resource limits, so that I can enjoy the world-building aspects of the game freely.

## Details

- **Creative Mode Features**:
  - Infinite resources (gold, materials, items)
  - No enemy aggression (monsters exist but are passive; can toggle hostility)
  - Fly mode: move through walls and fly over terrain; explore the entire map freely
  - Instant crafting: no skill requirements, no crafting time
  - God mode: invulnerable, infinite HP/mana; can toggle on/off
  - Time control: speed up, slow down, or pause the in-game clock
  - Spawn any creature or NPC at any location
  - Weather and time of day controls
- **Building Tools**:
  - Place and remove terrain tiles (walls, floors, water, lava, grass, etc.)
  - Build structures room by room
  - Furnish interiors with objects from the game's tile set
  - Create custom dungeons (like dungeon-sharing editor but available anywhere)
  - Paint tile colors for custom ASCII art
- **Experiment Lab**:
  - Test combat builds with adjustable dummy enemies (set level, type, abilities)
  - Preview spell effects and combos without resource cost
  - Test potion recipes, enchantments, and crafting without consuming materials
  - Simulate boss fights at any difficulty level
- **Creative Mode Saves**: separate from normal saves; cannot be converted to normal mode (prevents cheating)
- **Showcase**: export creative mode builds as shareable codes (same format as dungeon-sharing)
- **NPC Playground**: spawn any NPC and have LLM conversations without quest consequences
- Creative mode progress does not affect leaderboards, achievements, or legacy systems

## Acceptance Criteria

- [ ] All creative mode features toggle correctly
- [ ] Building tools allow placement and removal of all tile types
- [ ] Experiment lab simulates combat without resource consumption
- [ ] Creative saves are segregated from normal mode saves
- [ ] Exported builds are compatible with dungeon-sharing format
