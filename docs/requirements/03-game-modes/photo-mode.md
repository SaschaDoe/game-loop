# Photo Mode

As a player, I want a photo mode where I can capture ASCII art screenshots of interesting scenes, so that I can save and share memorable moments from my adventure.

## Details

- Pause the game and freely move the camera
- Toggle UI elements on/off
- Add a border and caption to the screenshot
- Export as a text file or image
- Gallery of saved screenshots accessible from the menu

## Acceptance Criteria

- [ ] Game pauses and camera is freely movable
- [ ] UI elements can be toggled
- [ ] Screenshots can be saved with captions
- [ ] Gallery displays saved screenshots
