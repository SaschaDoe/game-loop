# Humanoid Enemies

As a player, I want to fight intelligent humanoid enemies who use tactics, equipment, and abilities like I do, so that some encounters feel like fighting another player.

## Details

- **Bandits**: varying classes (archer, swordsman, mage), set ambushes on roads, demand gold before attacking
- **Cultists**: shadow magic users, summon demons, sacrifice captured NPCs
- **Mercenaries**: well-equipped fighters hired by rival factions, adapt tactics mid-fight
- **Rival Adventurers**: NPC parties with a tank, healer, and DPS; use potions and abilities
- **Assassins**: stealth approach, poison weapons, flee if detected early
- **Warlocks**: corrupted mages with powerful but unstable spells (chance to backfire and damage themselves)
- **Slavers**: capture instead of kill; if player loses, wake up in a slave camp (escape quest)
- Humanoid enemies can be: talked to (surrender option via LLM dialogue), pickpocketed, intimidated into fleeing
- Humanoid enemies use items: drink healing potions, throw bombs, use scrolls
- Some humanoid enemies have names and backstories discoverable through lore

## Acceptance Criteria

- [ ] All humanoid enemy types have distinct combat behaviors
- [ ] Humanoid enemies use items and abilities tactically
- [ ] Surrender/negotiation via LLM dialogue is possible
- [ ] Named humanoid enemies have discoverable backstories
