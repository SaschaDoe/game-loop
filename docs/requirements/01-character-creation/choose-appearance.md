# Choose Appearance

As a player, I want to customize my character's ASCII appearance, so that my character feels personal and recognizable in the game world.

## Details

- Choose the character symbol (e.g., @, &, $, or a letter)
- Choose the character color from a palette
- Optionally pick a title prefix displayed in dialogues (e.g., "Sir", "Shadow", "Elder")
- Appearance is shown in the HUD and during NPC conversations
- Other entities in the world have distinct appearances to avoid confusion

## Acceptance Criteria

- [ ] Player can select a symbol and color
- [ ] Selected appearance renders correctly on the map
- [ ] Appearance is referenced in dialogue UI
