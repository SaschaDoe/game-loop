# Choose Class

As a player, I want to choose a class for my character, so that I have a distinct playstyle with unique abilities and equipment proficiencies.

## Details

- Offer at least 8 classes: Warrior, Mage, Rogue, Ranger, Cleric, Paladin, Necromancer, Bard
- Each class has a starting equipment loadout
- Each class has a unique ability tree (see character-progression epic)
- Classes define which weapons and armor types are usable
- Some classes have alignment restrictions (e.g., Paladin must start Lawful Good)
- Display a short description of each class's playstyle

## Acceptance Criteria

- [ ] All classes are selectable
- [ ] Starting equipment is granted based on class
- [ ] Class-specific abilities are unlocked at level 1
- [ ] Class restrictions are enforced
