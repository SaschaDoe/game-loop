# Starting Equipment

As a player, I want my starting equipment to reflect my chosen class and backstory, so that my character feels personalized from the first moment of gameplay.

## Details

- **Class-based starter kits**:
  - **Warrior**: iron sword, wooden shield, leather armor, 2 health potions, 50 gold
  - **Mage**: wooden staff, cloth robe, spell scroll (Fireball), 3 mana potions, 30 gold
  - **Rogue**: dagger, lockpick set (5 uses), dark cloak (+stealth), smoke bomb, 40 gold
  - **Ranger**: shortbow, 20 arrows, leather armor, rope (10m), trail rations (3 days), 35 gold
  - **Cleric**: mace, holy symbol, chain mail, bandage kit, 2 holy water, 25 gold
  - **Bard**: lute, rapier, fancy clothes, charm scroll, wine bottle, 45 gold
- **Backstory modifiers** (added on top of class kit):
  - Noble: +100 gold, signet ring (opens noble dialogue options)
  - Urchin: lockpick set, street map (reveals hidden alleys), -20 gold
  - Soldier: military insignia, extra weapon, field rations
  - Scholar: 3 random lore books, quill and ink, magnifying glass
  - Criminal: disguise kit, stolen gem (valuable but flagged as stolen), fence contact
  - Hermit: herbalism kit, rare herb (3x), hand-drawn map to a secret location
- All starting items are slightly worn (lower durability than fresh items) to encourage early upgrades
- A tutorial chest near the starting area contains basic supplies the player may have missed

## Acceptance Criteria

- [ ] Each class receives correct starter kit
- [ ] Backstory modifiers add appropriate bonus items
- [ ] Starting items have reduced durability
- [ ] Tutorial chest provides supplementary basics
