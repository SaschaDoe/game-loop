# Choose Race

As a player, I want to choose my character's race from a variety of fantasy races, so that my character has unique racial traits and abilities that affect gameplay.

## Details

- Offer at least 6 races: Human, Elf, Dwarf, Orc, Halfling, Tiefling
- Each race provides stat bonuses (e.g., Elves get +2 Dexterity, Dwarves get +2 Constitution)
- Each race has a unique passive ability (e.g., Elves have darkvision, Dwarves have poison resistance)
- Display an ASCII art preview of the selected race
- Show a short lore paragraph for each race to help the player decide
- Race choice influences NPC dialogue and faction reputation

## Acceptance Criteria

- [ ] All races are selectable from the character creation screen
- [ ] Stat bonuses are applied correctly
- [ ] Racial passive abilities function in gameplay
- [ ] Race selection is saved to the character state
