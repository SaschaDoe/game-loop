# Choose Starting Alignment

As a player, I want to choose my character's starting moral alignment during creation, so that my initial disposition shapes early NPC reactions and available options.

## Details

- 9 alignments on the classic grid: Lawful Good, Neutral Good, Chaotic Good, Lawful Neutral, True Neutral, Chaotic Neutral, Lawful Evil, Neutral Evil, Chaotic Evil
- Starting alignment affects:
  - Initial faction reputation (Holy Order likes Good, Thieves Guild likes Chaotic)
  - Available starting quests (Good characters get rescue missions, Evil characters get sabotage jobs)
  - Some dialogue options are alignment-gated early on
  - Companion recruitment: some companions only join compatible alignments
- Alignment shifts freely during gameplay based on actions (see moral alignment epic)
- Certain classes have alignment restrictions: Paladin (must start Good), Necromancer (must start Evil/Neutral)
- Description of each alignment with example behavior for player guidance

## Acceptance Criteria

- [ ] All 9 alignments are selectable
- [ ] Starting reputation adjusts based on alignment
- [ ] Class alignment restrictions are enforced
- [ ] Early-game content varies by starting alignment
