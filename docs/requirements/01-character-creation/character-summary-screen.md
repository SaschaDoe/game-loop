# Character Summary Screen

As a player, I want to review a summary of all my character creation choices before starting the game, so that I can confirm or go back and change anything.

## Details

- Display name, race, class, ability scores, backstory, and appearance
- Show derived stats (HP, mana, armor, damage range)
- Allow going back to any previous step to make changes
- "Begin Adventure" button to finalize and enter the game world
- Save the character profile for potential future reuse or export

## Acceptance Criteria

- [ ] All character details are displayed on summary screen
- [ ] Player can navigate back to any creation step
- [ ] Finalizing starts the game with the correct character state
