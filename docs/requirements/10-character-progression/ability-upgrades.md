# Ability Upgrades

As a player, I want to upgrade my abilities to more powerful versions as I level up, so that my existing skills remain relevant and grow with me.

## Details

- Each active ability has 3 upgrade tiers (e.g., Fireball -> Greater Fireball -> Inferno)
- Upgrades increase damage, reduce cost, add effects, or change the ability fundamentally
- Upgrade requires: reaching a level threshold + spending skill points
- Some upgrades are mutually exclusive (choose to upgrade in one direction)
- Upgraded abilities have new visual effects in the ASCII combat display
- Passive abilities upgrade automatically with level milestones

## Acceptance Criteria

- [ ] Abilities can be upgraded through 3 tiers
- [ ] Mutually exclusive upgrade paths are enforced
- [ ] Upgraded abilities have improved stats and effects
- [ ] Visual changes reflect the upgrade
