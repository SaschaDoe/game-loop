# Skill Trees

As a player, I want to invest skill points in branching skill trees, so that I can specialize my character and develop unique builds.

## Details

- Each class has 3 skill tree branches (e.g., Warrior: Arms, Defense, Tactics)
- Skill points earned on level-up (1 per level)
- Skills have prerequisites: must invest in earlier skills before unlocking deeper ones
- Skills can be passive (stat boosts) or active (new abilities)
- Respec option available at a cost (gold or rare item) to reallocate points
- Visual tree display shows unlocked, available, and locked skills
- Synergies between trees reward hybrid builds

## Acceptance Criteria

- [ ] Skill trees display correctly with prerequisites
- [ ] Skill points can be invested and grant effects
- [ ] Active skills are usable in combat
- [ ] Respec functionality works
