# Prestige Classes

As a player, I want to unlock advanced prestige classes at high levels, so that long-term play is rewarded with powerful new specializations.

## Details

- Prestige classes unlock at level 20+ with specific prerequisites (stats, quests, faction rank)
- Examples: Warrior -> Warlord or Berserker, Mage -> Archmage or Spellblade, Rogue -> Assassin or Shadowdancer
- Prestige classes grant a new skill tree branch and exclusive abilities
- Changing to a prestige class is permanent and cannot be undone
- Prestige classes have unique visual flair (character symbol or color changes)
- Quest to unlock each prestige class involves proving mastery of your discipline

## Acceptance Criteria

- [ ] Prestige classes are unlockable at the correct level with prerequisites
- [ ] New skill tree and abilities are granted
- [ ] Class change is permanent
- [ ] Unlock quests are completable
