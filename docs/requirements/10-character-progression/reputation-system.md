# Reputation System

As a player, I want a reputation system that tracks how different groups perceive me, so that my actions have lasting social consequences.

## Details

- Reputation tracked per faction and per town
- Reputation levels: Hostile, Unfriendly, Neutral, Friendly, Honored, Revered
- Actions that affect reputation: quest completion, combat, theft, dialogue choices, gifts
- High reputation unlocks: discounts, exclusive quests, faction abilities, restricted areas
- Low reputation results in: higher prices, refused service, attacks on sight
- Reputation is displayed in a dedicated panel
- Some actions affect multiple reputations (helping one faction may anger their rival)

## Acceptance Criteria

- [ ] Reputation tracks across all factions and towns
- [ ] Reputation changes based on player actions
- [ ] Benefits and penalties apply at appropriate reputation levels
- [ ] Rival faction dynamics create reputation trade-offs
