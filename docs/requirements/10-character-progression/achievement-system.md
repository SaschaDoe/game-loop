# Achievement System

As a player, I want to unlock achievements for notable accomplishments, so that I have long-term goals and milestones to pursue beyond the main story.

## Details

- Achievement categories: combat, exploration, social, crafting, lore, challenge
- Examples: "Slay 100 goblins", "Discover all biomes", "Max out a skill tree", "Complete the game without dying", "Befriend every NPC in a town"
- Achievements grant titles that can be displayed on the character
- Some achievements unlock cosmetic rewards (character colors, symbols)
- Hidden achievements for secret discoveries
- Achievement progress is tracked and viewable in a menu

## Acceptance Criteria

- [ ] Achievements trigger when conditions are met
- [ ] Progress tracking updates correctly
- [ ] Rewards (titles, cosmetics) are granted
- [ ] Hidden achievements are revealed on completion
