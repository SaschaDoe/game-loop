# Crafting Skill Trees

As a player, I want dedicated skill trees for crafting professions, so that investing in crafting makes me progressively better and unlocks exclusive recipes.

## Details

- Crafting skill trees (separate from class combat trees):
  - **Smithing**: weapon/armor crafting quality, repair efficiency, masterwork items
  - **Alchemy**: potion potency, reduced ingredient waste, unlock rare recipes, dual-brew (2 potions at once)
  - **Enchanting**: stronger enchantments, reduced failure chance, dual enchantments on items
  - **Cooking**: better buff durations, unlock gourmet recipes, food never burns
  - **Trap Making**: stronger traps, more deployable simultaneously, invisible traps
  - **Runecraft** (new): inscribe runes on gear for passive effects (rune of speed, rune of thorns)
- Skill points earned through crafting actions (XP per craft), not from combat leveling
- Each tree has 3 tiers with 4-5 nodes per tier
- Tier 3 capstone talents are powerful: "Mastersmith" (crafted weapons have +2 rarity), "Philosopher's Stone" (transmute common ingredients into rare ones)
- Crafting skills are class-independent; any class can invest in any crafting tree

## Acceptance Criteria

- [ ] Crafting XP accrues from crafting actions
- [ ] Skill nodes unlock meaningful improvements
- [ ] Capstone talents provide powerful bonuses
- [ ] All classes can invest in crafting trees
