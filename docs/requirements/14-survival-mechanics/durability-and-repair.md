# Durability and Repair

As a player, I want my equipment to degrade with use and require repair, so that gear maintenance is a meaningful part of resource management.

## Details

- Weapons lose durability with each attack; armor loses durability when hit
- At 0 durability: weapons deal half damage, armor provides no protection
- Repair at blacksmith NPCs (costs gold) or with repair kits (consumable item)
- Higher-quality items degrade slower
- Enchanted items have a self-repair enchantment option
- Durability percentage shown on equipped items
- Broken item notification warns when durability is critical

## Acceptance Criteria

- [ ] Durability decreases with use
- [ ] 0-durability penalties apply correctly
- [ ] Repair restores durability via NPC or repair kit
- [ ] Durability display and warnings function
