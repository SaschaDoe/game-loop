# Aethermoor — ASCII RPG

## Project

SvelteKit/TypeScript ASCII RPG, statically hosted on Netlify.

- App: `ascii-rpg/`
- Requirements (epics & user stories): `docs/requirements/`
- Lore bible: `docs/lore/`

## Tech

Run from `ascii-rpg/`:
- `npm install` — install dependencies
- `npm run dev` — local dev server
- `npm test` — run tests

## Conventions

- Always run tests before committing. Always commit and push after completing work.
- Never break existing game systems when adding features.
- Keep the ASCII aesthetic consistent.

## Lore

The core twist: the seven gods were the worst mortals in history who stole divine thrones and hide their pasts. The lore layer (Ascended gods, Original Seven, factions, prophecies) must stay hidden and discoverable — never dump exposition on the player.
